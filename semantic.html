DETAILS AND SUMMARY TAGS

Which hides the details of an element inside a toggle,

which allows you to put a summary to appear on toggle element. The summary tag is the summary of the longer description. The summary element appears as a title for whatever you have in the detail. While the detail element is always hidden and can only be viewed when the toggle is opened.The detail tag is a detailed description of the summary.
