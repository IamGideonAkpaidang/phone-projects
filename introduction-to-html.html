    HOW TO USE THE IMAGE TAG

  • img - the image element
  • src - this is where we include the source of the image.
  • alt - the alt is where we include the alternative text that'll show in case the image doesn't load or open. It is often used to describe what the image is all about. The alt element can also be used to engage seo if you use relevant keywords.
